# python_basic_-_logic
This repository contains 10 beginner-friendly Python projects designed to help you practice basic programming concepts, arithmetic operations, conditional statements, and loops. 
Each project is implemented in a separate Python file with clear and simple code.

✨ About This Repository
This repository is part of a 50-question Python practice series.
It contains the first 10 problems, which belong to the Basic & Logic category.

✅concepts Covered
1. Arithmetic operations (+, -, *, /)
2. Conditional statements (if, elif, else)
3. Loops (for and while)
4. Basic input/output in Python
5. Simple problem-solving and logical thinking
